[폴더 구성]
  grade-analyzer/
    ├─ index.html          ← 프로그램 본체 (더블클릭 실행 가능)
    ├─ vendor/             ← 라이브러리 (완전 오프라인 동작)
    │   ├─ xlsx.full.min.js
    │   ├─ chart.umd.min.js
    │   ├─ plotly.min.js
    │   └─ html2pdf.bundle.min.js
    └─ README.txt          ← 본 문서


[실행 방법 2가지]

방법 A. 파일 더블클릭 (가장 간단)
  index.html 파일을 더블클릭하면 기본 브라우저에서 바로 열립니다.
  인터넷 연결이 필요 없습니다.

방법 B. 로컬 웹 서버 (권장, 엑셀 업로드 안정성 ↑)
  해당 폴더에서 아래 중 하나를 실행:
    • Node.js 있을 때:  npx http-server . -p 8091
    • Python 있을 때:    python -m http.server 8091
  그다음 브라우저에서  http://localhost:8091  접속


[다른 PC로 이동하는 법]

1. 프로그램 자체 옮기기
   이 grade-analyzer 폴더 전체를 USB·외장하드·공유폴더로 복사.
   → 새 PC에서 위 "실행 방법"으로 열면 끝.

2. 분석 데이터 함께 옮기기 (3가지 중 선택)

   방법 1: 백업 파일 (권장)
     ① 프로그램 내에서 Ctrl+S → "📤 현재 분석 내보내기" 클릭
     ② 생성된 .json 파일을 USB 등으로 복사
     ③ 새 PC의 프로그램에서 Ctrl+S → "📥 백업 불러오기" → 파일 선택

   방법 2: 전체 백업
     ① Ctrl+S → "📦 전체 백업" (저장된 모든 프로젝트를 하나의 파일로)
     ② 새 PC에서 "📥 백업 불러오기"로 복원

   방법 3: 원본 Excel 파일 재업로드
     백업 없이 원본 .xlsx 파일만 새 PC에서 다시 업로드.


[데이터 보관 위치]
  • 업로드한 엑셀 데이터 → 브라우저 메모리에만 있음 (창 닫으면 사라짐)
  • Ctrl+S 로 저장한 프로젝트 → 브라우저 LocalStorage (PC별·브라우저별로 분리)
  • 백업 파일(.json) → 사용자가 지정한 다운로드 폴더

  ⚠ 브라우저 캐시/기록을 삭제하면 LocalStorage도 함께 지워집니다.
     중요한 데이터는 반드시 📤/📦 로 .json 백업을 만들어 두세요.


[데이터 프라이버시]
  이 프로그램은 어떠한 데이터도 인터넷으로 전송하지 않습니다.
  모든 분석은 브라우저 내 JavaScript로만 처리됩니다.
  vendor/ 폴더의 라이브러리도 로컬에서 로드하므로 CDN 연결이 불필요합니다.
