프로젝트 관리

현재 분석 데이터·업로드 파일·필터·차트가 모두 비워지고 업로드 화면으로 돌아갑니다. 내 차트/내 그룹/매핑 양식은 보존됩니다.
백업 파일(.json)을 USB·이메일·클라우드 드라이브 등으로 옮겨 다른 PC의 이 프로그램에서 "백업 불러오기"로 이어 작업할 수 있습니다.

분석 자유도 (NEW)

  • 드릴다운 — 거의 모든 차트의 막대/점을 클릭하면 해당 그룹의 학생/과목 명단 모달이 열립니다. 모달 우측 상단 CSV 버튼으로 전체 내보내기.
  • 교차 탐색 (탭 7) — 행/열/측정값을 자유롭게 조합한 히트맵. 셀 클릭 → 해당 그룹 드릴다운.
  • 내 그룹 — 사이드바 그룹 관리에서 학생을 묶어 저장. 그룹 칩 클릭 시 분석 전체가 그 그룹으로 좁혀집니다 (필터처럼 동작).

자동 저장

업로드한 데이터는 LocalStorage에 자동 저장되어 새로고침이나 브라우저 재시작 후에도 복원됩니다. 첫 화면의 이전 작업 이어하기 카드를 클릭하면 즉시 복원됩니다. 내 그룹은 분석 데이터와 분리되어 별도로 저장되므로 새 데이터를 올려도 유지됩니다.

데이터 형식

모의고사: 학번, 이름, 반, 회차, 과목, 원점수, 표준점수, 백분위, 등급

내신: 학번, 이름, 반, 학기, 과목, 단위수, 원점수, 과목평균, 표준편차, 석차등급
